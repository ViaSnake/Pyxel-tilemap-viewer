# Pyxel tilemap viewer
Pyxel tilemap viewer is a dedicated tilemap display program for tilemap design.

# How to use
1. Download `viewr.py`.
2. Place tm.pyxres in the same directory as `viwer.py`.
3. Run viewer.py to see the tilemap.

W: Move up
S: Move down
A: Move left
D: Move right

0~7: Change tilemap
